### 使用须知：

1. 环境
   1. node，gulp，npm
2. git 仓库下载 （git clone https://github.com/zhaojianing/gulp）
3. 初始化本地 npm init    或者（npm init -y）
   1. 如果有问题可以看文件创建须知文件（已采坑）
4. 下载依赖 npm install
5. 更改package.json文件或者如下更改
